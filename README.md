# 8-bit-updowncounter
Developed an 8-bit synchronous Up/Down Counter in VHDL using Xilinx Vivado, including design, simulation, synthesis, and FPGA implementation for digital system applications.

## Features
- Up counting
- Down counting
- Synchronous operation
- Implemented on FPGA
  
- ## Tools Used
- Vivado
- VHDL
- ZedBoard
